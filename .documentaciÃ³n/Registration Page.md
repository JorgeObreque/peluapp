## Validations

![image-20230614225544697](/home/jorge/Imágenes/image-20230614225544697.png)

**Description**

This page allows to users create an account

**Fields**

- Nombre
  - min length 3
  - max length 15 
  - only letters
  - not null
- Email
  - Unique
  - Email type
  - max length 30
  - not null
- Contraseña
  - min length 6
  - max length 15
  - equals repetir contraseña
  - not null 
- Repetir Contraseña
  - equals contraseña
  - not null
